Welcome to NetHealth's Documentation
==================================

NetHealth is a comprehensive network analysis tool that helps you monitor and diagnose network conditions.

Quick Start
----------

Install NetHealth::

    pip install nethealth

Run a network analysis::

    nethealth analyze

View historical data::

    nethealth history

Features
--------

* Network speed testing
* Network fingerprinting
* VPN detection
* Interface monitoring
* Historical tracking
* Plugin system
